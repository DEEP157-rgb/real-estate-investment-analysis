# Real Estate Investment Analysis & Property Valuation Model

## Overview
This project analyzes real estate properties using financial and data science techniques to support investment decisions.

## Key Features
- Data Cleaning & EDA
- Feature Selection
- ROI Calculation
- Cap Rate Analysis
- DSCR (Debt Service Coverage Ratio)
- Comparable Sales Analysis
- Market Trend Analysis
- Investment Grading (A/B/C)
- Machine Learning Model (Price Prediction)

## Business Use Case
- Property valuation
- Investment screening
- Deal analysis
- Portfolio decision-making

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
